Mobile Smile (Sistem Manajemen Kunjungan Dealer)

Mobile Smile adalah aplikasi mobile yang dirancang untuk mempermudah dan mendokumentasikan proses kunjungan sales/marketing ke dealer. Aplikasi ini mendukung fitur-fitur seperti pencatatan kunjungan langsung (Direct Visit), pengelolaan tugas kunjungan (Task Visit), dan riwayat kunjungan (History Visit), serta data master yang relevan untuk PT Suzuki Finance Indonesia.

Proyek ini merupakan bagian dari pengembangan internal di PT Suzuki Finance Indonesia, dengan fokus pada peningkatan efisiensi operasional kunjungan lapangan.

  • Peran: Mobile Developer
  • Kontribusi:
    • Mengembangkan dan memelihara fitur-fitur utama aplikasi seperti Direct Visit, Task Visit, dan History Visit menggunakan Flutter.
    • Berperan aktif dalam membangun antarmuka pengguna yang intuitif dan responsif.
    • Melakukan integrasi API dengan backend untuk sinkronisasi data kunjungan dan dealer.
    • Memastikan fitur lokasi berfungsi dengan akurat untuk pencatatan kunjungan.
  • Teknologi: Flutter, Backend (tidak spesifik, kemungkinan Laravel/lainnya), MySQL (untuk database), GitHub
  • Fitur: Direct Visit, Task Visit, History Visit, Data Master, Geolocation Tracking, Photo Upload

Detail Proyek Mobile Smile

🎯 Tujuan Sistem

  • Mencatat dan mendokumentasikan setiap kunjungan ke dealer secara digital dan terstruktur.
  • Mempermudah perencanaan dan pelaksanaan tugas kunjungan sales/marketing.
  • Menyediakan riwayat kunjungan yang lengkap untuk analisis dan pelaporan.
  • Meningkatkan akurasi data lokasi kunjungan melalui integrasi GPS.
  • Memfasilitasi pengunggahan bukti foto kunjungan di lokasi.

🛠 Fitur Utama

  • Direct Visit: Form untuk mencatat kunjungan langsung dengan detail PIC, kontak, tujuan, tema diskusi, dan lokasi GPS.
  • Task Visit: Mengelola daftar tugas kunjungan yang harus dilakukan (jika ada data).
  • History Visit: Melihat riwayat lengkap semua kunjungan yang telah dilakukan, termasuk detail kunjungan dan foto.
  • Data Master: Mengakses data master seperti jabatan, area, cabang, produk, dan dealer.
  • Upload Foto: Fitur untuk mengunggah foto sebagai bukti kunjungan.
  • Lokasi GPS: Otomatis mendeteksi dan menyimpan koordinat lokasi kunjungan.

💻 Teknologi yang Digunakan

  • Frontend: Flutter
  • Backend: Laravel
  • Database: MySQL (sesuai dengan setup umum untuk Laravel)
  • Version Control: Git / GitHub

👩‍💻 Peran Saya

Sebagai Mobile Developer dalam pengembangan aplikasi Mobile Smile di PT Suzuki Finance Indonesia, saya bertanggung jawab penuh dalam membangun dan memelihara antarmuka pengguna serta logika fungsionalitas aplikasi. Saya fokus pada implementasi fitur-fitur penting seperti pencatatan kunjungan, tampilan riwayat, dan integrasi dengan layanan lokasi serta kamera.

  • Mengimplementasikan dan mengembangkan modul Direct Visit, Task Visit, dan History Visit.
  • Mengintegrasikan API untuk mengambil dan mengirim data kunjungan ke backend.
  • Melakukan debugging dan optimasi performa aplikasi.

📌 Manfaat Proyek

Aplikasi Mobile Smile telah secara signifikan meningkatkan efisiensi proses kunjungan dealer di PT Suzuki Finance Indonesia. Dengan digitalisasi pencatatan, perusahaan dapat memantau aktivitas lapangan secara real-time, memastikan data kunjungan yang akurat, dan mempercepat proses pelaporan. Hal ini membantu manajemen dalam mengambil keputusan strategis berdasarkan data kunjungan yang valid dan komprehensif.
