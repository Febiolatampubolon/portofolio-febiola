Perjalanan Menjadi Front-End Developer

Dipublikasikan oleh Febiola Tampubolon • 24 Maret 2025

Saat pertama kali saya mulai belajar pemrograman, semuanya terasa sangat membingungkan. HTML dan CSS tampak sederhana, tapi ketika mulai mencoba membuat layout sendiri, saya sadar bahwa dunia front-end development itu luas dan penuh tantangan.

Saya memulai dari yang paling dasar: membuat halaman statis dengan HTML dan CSS. Saat itu, saya belum tahu apa itu responsive design, grid system, atau bahkan positioning. Namun seiring waktu, saya terus mencoba, menonton tutorial, membaca dokumentasi, dan tentu saja—banyak mencoba dan gagal.

Setelah memahami dasar, saya mulai menggunakan Tailwind CSS, sebuah framework CSS yang benar-benar mengubah cara saya membuat antarmuka. Utility-first approach dari Tailwind membuat saya lebih cepat mendesain UI yang konsisten dan responsive.

Lalu saya lanjut ke dunia JavaScript dan React. Awalnya menakutkan, tapi begitu saya mengerti komponen, props, dan state, saya mulai menikmati membangun UI dinamis. Saya pun membuat berbagai mini project sebagai latihan.

Akhirnya, saya berhasil membuat portfolio website saya sendiri. Desainnya simple dan profesional, dengan bagian tentang saya, project, blog, dan kontak. Hal ini menjadi milestone penting dalam perjalanan saya sebagai seorang front-end developer.
