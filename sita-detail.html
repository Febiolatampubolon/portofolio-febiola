SITA (Sistem Informasi Tugas Akhir)

SITA adalah platform berbasis web untuk mengelola proses tugas akhir mahasiswa, mulai dari pendaftaran, bimbingan, seminar, hingga sidang. Sistem ini membantu mempercepat dan merapikan alur kerja antara mahasiswa dan dosen pembimbing.

  • Peran: System Analyst, Frontend Developer, Scrum Master
  • Kontribusi:
    • Menganalisis kebutuhan pengguna dan menyusun dokumentasi fungsional.
    • Mendesain serta mengembangkan antarmuka frontend dengan React dan Tailwind CSS.
    • Memimpin tim menggunakan metode Agile Scrum dengan sprint mingguan dan daily meeting.
  • Teknologi: ReactJS, Tailwind CSS, Vite, REST API, GitHub
  • Fitur: Pendaftaran TA, Log Bimbingan, Penjadwalan Seminar & Sidang, Validasi Dokumen
