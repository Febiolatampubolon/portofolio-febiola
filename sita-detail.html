SITA (Sistem Informasi Tugas Akhir)

SITA adalah platform web untuk mengelola proses tugas akhir mahasiswa, dari pendaftaran, log bimbingan, seminar, hingga sidang. Aplikasi ini mempercepat dan merapikan alur kerja akademik tingkat akhir.

Sistem ini dikembangkan untuk lingkungan akademik Institut Teknologi Del dan mendukung pengarsipan serta pemantauan kegiatan mahasiswa akhir.

  • Peran: Developer dan System Analyst
  • Kontribusi:
    • Mengimplementasikan analisis kebutuhan pengguna dan menyusun dokumentasi sistem seperti alur, use case, dan spesifikasi teknis pada proyek SITA.
    • Mengembangkan antarmuka pengguna proyek SITA menggunakan ReactJS dan Tailwind CSS, serta memastikan desain responsif dan mudah digunakan di berbagai perangkat.
    • Mengintegrasikan frontend dengan REST API Laravel untuk mendukung proses pendaftaran TA, log bimbingan, hingga penjadwalan seminar dan sidang.
  • Teknologi: ReactJS, Tailwind CSS, Laravel, MySQL, GitHub
  • Fitur: Pendaftaran TA, Log Bimbingan, Jadwal Seminar & Sidang, Validasi Dokumen
