SIMBA (Sistem Informasi Monitoring Bimbingan Akademik)

SIMBA merupakan aplikasi berbasis web yang dirancang untuk membantu proses perwalian, konseling, dan pemantauan pelanggaran mahasiswa. Proyek ini dikembangkan sebagai solusi digital terintegrasi bagi bagian Kemahasiswaan dan Dosen Wali.

  • Peran: System Analyst, Frontend Developer
  • Kontribusi:
    • Menyusun dokumentasi teknis (SDD) dari hasil analisis kebutuhan.
    • Membuat desain UI untuk fitur perwalian, konseling, dan pelanggaran.
    • Mengembangkan antarmuka frontend dan mengintegrasikan dengan REST API.
  • Teknologi: Flutter, Laravel, MySQL, GitHub
  • Fitur: Monitoring Konseling, Pelanggaran Mahasiswa, Status & Jadwal Perwalian, Berita Acara
