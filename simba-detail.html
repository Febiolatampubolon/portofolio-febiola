SIMBA (Sistem Informasi Monitoring Bimbingan Akademik)

SIMBA merupakan aplikasi berbasis web yang dirancang untuk membantu proses perwalian, konseling, dan pemantauan pelanggaran mahasiswa.

  • Peran: System Analyst, Developer
  • Kontribusi:
    • Menganalisis kebutuhan fungsional dan menulis dokumentasi teknis (SDD).
    • Merancang dan mengembangkan antarmuka pengguna untuk fitur-fitur SIMBA.
    • Mengintegrasikan antarmuka dengan backend menggunakan REST API Laravel.
  • Teknologi: Flutter, Laravel, MySQL, GitHub
  • Fitur: Monitoring Konseling, Pelanggaran Mahasiswa, Status & Jadwal Perwalian, Berita Acara

Detail Proyek SIMBA

🎯 Tujuan Sistem

  • Meningkatkan efektivitas proses perwalian dan konseling.
  • Menyediakan dokumentasi digital riwayat pelanggaran dan hasil konseling.
  • Mempermudah pengelolaan informasi mahasiswa oleh dosen dan bagian kemahasiswaan.

🛠️ Fitur Utama

  • Melihat status dan jadwal perwalian mahasiswa.
  • Melihat berita acara bimbingan dan riwayat pertemuan.
  • Manajemen pelanggaran dan riwayat konseling mahasiswa.
  • Request dan pengelolaan sesi konseling antara mahasiswa dan konselor.
  • Upload hasil konseling, notifikasi otomatis, dan dashboard akademik.

💻 Teknologi yang Digunakan

  • Frontend: Flutter
  • Backend: Laravel (PHP)
  • Database: MySQL

👩‍💻 Peran Saya

Dalam proyek ini, saya bertanggung jawab sebagai System Analyst dan Developer.

  • Melakukan analisis kebutuhan sistem dan menyusun SDD secara lengkap.
  • Mengembangkan UI menggunakan Flutter dan menghubungkannya dengan API Laravel.
  • Berkoordinasi dengan tim backend dan QA untuk memastikan semua fitur berjalan baik.

📌 Manfaat Proyek

Dengan SIMBA, proses perwalian, konseling, dan pemantauan pelanggaran mahasiswa menjadi lebih terdokumentasi, transparan, dan dapat dipantau secara real-time. Sistem ini mendukung lingkungan kampus yang lebih terstruktur dan mendukung perkembangan mahasiswa.
